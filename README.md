# Ghetto Gist Manager

## Prerequisites
- GitHub Client (gh)
- fzf
- nano

## Description

This is a simple script that allows you to manage your gists from the command line. It is written in Bash and uses 'gh' to interact with the GitHub API.

## Installation
```Bash
git clone <the repo>
cd <the repo>
chmod +x gist.sh
sudo cp gist.sh /usr/bin/gist
```

## Usage
```Bash
gist
```
